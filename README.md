# Period Tracker App

A Flutter starter app for tracking menstrual cycles.

## Features

- Enter last period start date.
- Track period length and average cycle length.
- Predict next period, ovulation, fertile window, and current cycle day.
- Log flow level, symptoms, mood, and notes.
- View a simple month calendar with period and fertile-day highlights.
- See care tips and a health disclaimer.

## Run

Open this folder in Android Studio:

```powershell
C:\Users\Awa Nisan Tojah\StudioProjects\period_tracker_app
```

Then run:

```powershell
flutter pub get
flutter run
```

Predictions are estimates based on the data entered by the user and should not be used as medical advice or contraception. This first version keeps entries during the app session only, so persistent storage can be added as the next feature.
